# My Claude Skills

租寓集團 Claude Code Skills 集合。
包含 Ragic 自動化流程、月報更新、滿房率登記等作業用 Skills。

## Skills 清單

| Skill | 說明 |
|-------|------|
| ragic-occupancy-registration | 租寓（大台北）當下間數與滿房率登記 |
| ragic-tainan-occupancy-registration | 好好租（台南）當下間數與滿房率登記 |
| ragic-taoyuan-occupancy-registration | 租寓（桃園）當下間數與滿房率登記 |
| ragic-taichung-occupancy-registration | 租寓（台中）當下間數與滿房率登記 |
| ragic-weekly-ad-transfer | 廣告來案週報拋轉 |
| ragic-property-management-monthly | 代管營業額(扣稅)月報更新 |
| ragic-rental-revenue-monthly | 代租營業額月報更新 |
| ragic-decoration-revenue-monthly | 裝潢/裝修營業額月報更新 |
| ragic-social-housing-income-monthly | 社宅收入月報更新 |
| ragic-payroll-monthly-update | 租顧個人獎金計算表每月更新 |
| robyn-monthly-report | Robyn 包租代管表每月更新與收益報告 |
| monthly-revenue-all | 每月四大營業額登記（代管+社宅+代租+裝修） |

## 安裝方式

```bash
git clone https://github.com/godorzme/my-claude-skills ~/.claude/skills/my-claude-skills
```
